EscapeSequence ::
  - CharacterEscapeSequence `/^[^\n\r\u2028\u2029\dxu]$/`
  - 0 \[lookahead ∉ DecimalDigit\]`/^0(?!=\d)$/`

  - UnicodeEscapeSequence`/^u([\da-fA-F]{4}|\{(10|0?[\da-fA-F])[\da-fA-F]{0,4}\})$/`
